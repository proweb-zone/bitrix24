Установите Bitrix24:

Откройте браузер и перейдите по адресу http://localhost

Следуйте инструкциям установки Bitrix24

В настройках базы данных укажите:

Сервер: mysql

Пользователь: bitrix_user

Пароль: bitrix_password

База данных: bitrix

Дополнительные команды:

Просмотр логов: docker-compose logs -f

Остановка: docker-compose down

Перезагрузка: docker-compose restart
